# @heise/eslint-config

The Heise eslint configuration.

```sh
$ cd my-project
$ yarn add --dev @heise/eslint-config # Or with npm
$ npm install --save-dev @heise/eslint-config
```

Edit `my-project/.eslintrc.js`:

```js
module.exports = {
  extends: '@heise',
  // Optional: Other configurations, for example:
  // parser: 'babel-eslint',
}
```
